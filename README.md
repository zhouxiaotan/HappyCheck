# HappyCheck
使用CppCheck进行静态代码检查的工具

使用方法：<br>
1.从下面的链接下载安装文件，进行安装。<br>
2.安装后在桌面点击HappyCheck图标，启动程序。<br>
3.选择源码目录并添加需要检查的包号，包号没有的时候，显示全部检查结果。<br>
4.点击Export导出结果到Excel中。<br>
